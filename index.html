Projetos.

O que eu já fiz?

Magic Coding.

O Magic Coding é um projeto desenvolvido em parceria com a designer Isabella Marques. Trata-se de um site que simula a venda de cursos voltados para a área de tecnologia da informação. Com um design chamativo e intuitivo, o objetivo é atrair novos assinantes para os cursos oferecidos.

Hemoglobina.

O Hemoglobina é um projeto desenvolvido em equipe como trabalho de conclusão de curso. É um aplicativo móvel que facilita a conexão entre doadores de sangue e hemocentros, abordando diretamente problemas que dificultam o aumento das doações, como a falta de conhecimento e a dificuldade em encontrar locais adequados para doar.

Preservateen.

O PreservaTeen é um projeto desenvolvido em equipe com o objetivo de desmistificar tabus, fornecer informações confiáveis e conscientizar sobre a importância do uso de preservativos para prevenir ISTs e gravidez indesejada entre adolescentes. O projeto promove decisões saudáveis e responsáveis.

InAlert.

O InAlert é um projeto que inova no campo da detecção de gases. Trata-se de um detector móvel de gases inflamáveis, desenvolvido em sistema embarcado e conectado a um aplicativo móvel. O sistema permite monitorar os gases de forma prática e eficiente.

RTS Engenharia Elétrica.

O RTS Engenharia Elétrica é um website desenvolvido para divulgar os serviços de uma empresa de engenharia. Além de ser uma vitrine para os trabalhos realizados, a plataforma também tem como objetivo captar novos clientes.

Triágil.

O Triágil é um sistema desenvolvido em equipe durante uma maratona de programação. Ele organiza informações do usuário, como identificação e convênio, facilita o acesso à carteirinha e aplica o protocolo de Manchester para priorizar atendimentos, agilizando o processo e reduzindo filas no pronto atendimento.
